## 🛠️ Architecture Notes

### Python Bridge (Chaquopy)
- **Engine**: `PythonBridge.kt` communicates with `download_bridge.py`.
- **Binaries**: Native `libffmpeg.so` and `libffprobe.so` must reside in `app/src/main/jniLibs/arm64-v8a/`.
- **Tracking**: `.gitignore` is configured to allow `.so` files within `jniLibs`.

### Storage Flow
1. **Workspace**: `Internal Storage/Android/data/com.universaldownloader/cache/download_work` (Temporary).
2. **Publishing**: Files are moved to `Downloads/Universal Downloader` via MediaStore.
3. **Cleanup**: Source files are strictly deleted only after successful system registration.

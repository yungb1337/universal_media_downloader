## 🛠️ Architecture Notes

### Python Bridge (Chaquopy)
The app uses Chaquopy to run a Python backend (`yt-dlp`). 
- **Bridge**: `PythonBridge.kt` calls `download_bridge.py`.
- **FFmpeg**: Must be placed in `jniLibs` and renamed to `libffmpeg.so` for Android to load it correctly.

### Storage & Permissions
- Uses Scoped Storage (SAF) for user-selected download directories.
- Foreground Service (`DownloadService.kt`) handles long-running downloads.
